Bienvenido a la Aplicación GitOps

Acerca de

Esta es una aplicación gestionada con GitOps. Se despliega y mantiene automáticamente utilizando ArgoCD, que sincroniza los manifiestos de Kubernetes almacenados en un repositorio Git.

Características

  • Despliegue automático desde Git
  • Sincronización constante con el repositorio
  • Escalabilidad y alta disponibilidad
  • Monitoreo y recuperación automática con ArgoCD
